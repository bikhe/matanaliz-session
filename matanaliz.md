# Шпора по математическому анализу

## 1. Функция, область определения, способы задания. Элементарные функции. Сложные и обратные функции

**Функция** — правило, по которому каждому x из области определения D(f) ставится в соответствие единственное значение y = f(x).

**Область определения** D(f) — множество всех x, при которых функция определена.

**Способы задания**:
- аналитический (формулой),
- табличный,
- графический,
- словесный.

**Основные элементарные функции**:
- постоянная y = c,
- линейная y = kx + b,
- квадратичная y = ax² + bx + c,
- степенная y = x^r (r рациональное),
- показательная y = a^x (a > 0, a ≠ 1),
- логарифмическая y = log_a x (a > 0, a ≠ 1),
- тригонометрические: sin x, cos x, tan x, cot x.

**Сложная функция** — композиция y = f(g(x)).  
D(f ∘ g): x ∈ D(g) и g(x) ∈ D(f).

**Обратная функция** y = f⁻¹(x) существует, если f строго монотонна.  
D(f⁻¹) = E(f), E(f⁻¹) = D(f).  
График — симметрия относительно y = x.

## 2. Числовая последовательность. Предел последовательности

**Числовая последовательность** — бесконечный упорядоченный набор чисел {x_n}, n = 1, 2, 3, ...

**Роль в вычислениях** — приближённое вычисление чисел (π, e), корней, значений функций через итерации и частичные суммы.

**Предел последовательности**  
lim (n→∞) x_n = a, если для любого ε > 0 существует N(ε) такое, что при n > N выполняется |x_n - a| < ε.  

Сходящаяся — имеет конечный предел.  
Расходящаяся — не имеет или ±∞.

## 3. Предел функции

**По Коши** (x → a):  
lim (x→a) f(x) = A, если ∀ ε > 0 ∃ δ > 0: при 0 < |x - a| < δ ⇒ |f(x) - A| < ε.

**По Гейне**:  
Для любой {x_n} → a (x_n ≠ a) последовательность {f(x_n)} → A.

**При x → ∞**:  
lim (x→∞) f(x) = A, если ∀ ε > 0 ∃ M: при x > M ⇒ |f(x) - A| < ε.

**Монотонная функция**:  
Если f монотонна и ограничена на (a, b), то существуют lim (x→a+) f(x) и lim (x→b-) f(x).  
Аналогично при x → ±∞.

## 4. Основные теоремы о пределах

- Единственность предела.
- Предел постоянной равен ей самой.
- Предел суммы, разности, произведения, частного (если знаменатель ≠ 0).
- Предел сложной функции: lim f(g(x)) = f(lim g(x)), если f непрерывна.
- Теорема о сохранении знака.
- Теорема о сжатии (зажатии).

**Пример**: lim (x→0) (sin x)/x = 1 (первый замечательный предел — следствие теоремы о сжатии).

## 5. Раскрытие неопределённостей

**Типы**: 0/0, ∞/∞, ∞ - ∞, 0 · ∞, 1^∞, 0^0, ∞^0.

**Правила**:
- 0/0, ∞/∞ — правило Лопиталя.
- ∞ - ∞ — привести к общему знаменателю.
- 0 · ∞ — привести к 0/0 или ∞/∞.
- 1^∞, 0^0, ∞^0 — логарифмирование.

**Пример**: lim (x→0+) x ln x = 0  
x ln x = (ln x)/(1/x) → (-∞)/∞ → Лопиталя → -x → 0.

## 6. Замечательные пределы

**Первый**: lim (x→0) (sin x)/x = 1.

**Второй**: lim (x→∞) (1 + 1/x)^x = e.

**Следствия**:
- lim (x→0) (tan x)/x = 1,
- lim (x→0) (arcsin x)/x = 1,
- lim (x→0) (1 + x)^(1/x) = e,
- lim (x→0) (a^x - 1)/x = ln a.

## 7. Эквивалентные бесконечно малые (при x → 0)

| Бесконечно малая   | Эквивалентная       |
|--------------------|---------------------|
| sin x              | x                   |
| tan x              | x                   |
| arcsin x           | x                   |
| arctan x           | x                   |
| 1 - cos x          | x²/2                |
| a^x - 1            | x ln a              |
| ln(1 + x)          | x                   |
| e^x - 1            | x                   |

**Пример**: lim (x→0) (tan x - sin x)/x³ = 1/2  
(используя tan x ~ x + x³/3, sin x ~ x - x³/6).

## 8. Непрерывность. Точки разрыва

**Непрерывность в x₀**:
1. f(x₀) определена,
2. Существуют односторонние пределы,
3. lim (x→x₀) f(x) = f(x₀).

**Точки разрыва**:
- I рода — конечные односторонние пределы (скачок).
- II рода — хотя бы один предел ∞ или не существует.

## 9. Неявно заданная функция

F(x, y) = 0.  
y'_x = - (F'_x)/(F'_y) (при F'_y ≠ 0).

**Пример**: x² + y² = 1 → y' = -x/y.

## 10. Параметрически заданная функция

x = φ(t), y = ψ(t).  
y'_x = (ψ'(t))/(φ'(t)) (при φ'(t) ≠ 0).

**Пример**: x = cos t, y = sin t → y'_x = -cot t.

## 11. Логарифмическое и показательно-степенное дифференцирование

**Логарифмическое**: y' = y · (ln |f(x)|)'.

**Показательно-степенная** y = f(x)^g(x):  
y' = y (g' ln f + g (f'/f)).

**Пример**: y = x^x → y' = x^x (ln x + 1).

## 12. Производная: определение и смысл

**Определение**: f'(a) = lim (x→a) (f(x) - f(a))/(x - a).

**Смысл**: скорость изменения; коэффициент наклона касательной.

**Пример**: f(x) = x², f'(1) = 2.

## 13. Производные высших порядков

f''(x) = (f'(x))', f^(n)(x) = (f^(n-1)(x))'.

**Пример**: y = x⁴ → y^(4) = 24.

## 14. Дифференциал

dy = f'(x) dx.

**Смысл**: приращение касательной.

**Пример**: √4.1 ≈ 2 + 0.025 = 2.025.

## 15. Касательная и нормаль

**Касательная**: y - y₀ = f'(x₀)(x - x₀).

**Нормаль**: y - y₀ = -1/f'(x₀)(x - x₀).

**Пример**: y = x², (1;1) → касательная y = 2x - 1.

## 16. Правила дифференцирования и таблица

**Правила**:
- (u ± v)' = u' ± v',
- (c u)' = c u',
- (u v)' = u' v + u v',
- (u/v)' = (u' v - u v') / v².

**Таблица производных**:

| f(x)      | f'(x)              |
|-----------|--------------------|
| c         | 0                  |
| x^n       | n x^{n-1}          |
| a^x       | a^x ln a           |
| e^x       | e^x                |
| ln x      | 1/x                |
| log_a x   | 1/(x ln a)         |
| sin x     | cos x              |
| cos x     | -sin x             |
| tan x     | 1/cos² x           |
| cot x     | -1/sin² x          |

## 17. Производная сложной и обратной функции

**Сложная**: (f(g(x)))' = f'(g) · g'.

**Обратная**: (f⁻¹(x))' = 1/f'(y).

## 18. Теоремы Ферма, Ролля, Лагранжа, Коши

**Ферма**: экстремум ⇒ f'(x₀) = 0.

**Ролля**: f(a)=f(b), непрер. на [a,b], дифф. на (a,b) ⇒ ∃c: f'(c)=0.

**Лагранжа**: ∃c: f'(c) = (f(b)-f(a))/(b-a).

**Коши**: (f(b)-f(a))/(g(b)-g(a)) = f'(c)/g'(c).

## 19. Правило Лопиталя

Для 0/0, ∞/∞:  
lim f/g = lim f'/g' (если существует).

**Условия**: дифференцируемы, пределы числ./знам. 0 или ∞, g' ≠ 0.

**Пример**: lim (x→0) (sin x)/x = 1.

## 20. Монотонность и экстремум

**Монотонность**:
- возрастает, если f' ≥ 0,
- убывает, если f' ≤ 0.

**Экстремум**:
- Необходимое: f'(x₀) = 0.
- Достаточное: смена знака f'.

## 21. Выпуклость и точка перегиба

**Выпуклость вверх**: f'' ≥ 0.  
**Выпуклость вниз**: f'' ≤ 0.

**Точка перегиба**: f''(x₀)=0 и смена знака f''.

## 22. Исследование функции и построение графика

**Этапы**:
1. Область определения.
2. Чётность, периодичность.
3. Разрывы, односторонние пределы.
4. Пересечение с осями.
5. f': критические точки, монотонность.
6. Экстремумы.
7. f'': выпуклость, перегибы.
8. Асимптоты.
9. Построение графика.

**Пример**: y = x³ - 3x — кубическая парабола с лок. максимумом и минимумом.
