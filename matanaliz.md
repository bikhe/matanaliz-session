1. Поясните понятие функции, области ее определения, способы ее задания. Основные элементарные функции, их свойства и графики. Сложные и обратные функции.

    **Функция** — соответствие, при котором каждому x из области определения D(f) ставится в соответствие единственное y = f(x).

    **Область определения** D(f) — множество значений x, при которых функция определена.

    **Способы задания**:
        - аналитический (формулой),
        - табличный,
        - графический,
        - словесный.

    **Основные элементарные функции**:
        - постоянная  y = c
        - линейная y = kx + b,
        - квадратичная y = ax^2 + bx + c,
        - степенная y = x^r,
        - показательная y = a^x (a > 0, a != 1),
        - логарифмическая (y = log_a(x) ) ( a > 0, a != 1 ),
        - тригонометрические: (sin x, cos x, tg x, ctg x).

    **Сложная функция** — композиция (f(g(x))).  
        Область определения: x принадлежит D(g) и g(x) принадлежит D(f).

    **Обратная функция** f^-1 существует, если f строго монотонна.  
        D(f^-1) = E(f), E(f^-1) = D(f).  
        График — симметричен графику f относительно прямой y = x.  
            f(f^-1(x)) = x, f^-1(f(x)) = x.

2. Поясните понятие числовой последовательности, их роль в вычислительных процессах. Предел числовой последовательности.
    
    **Числовая последовательность** — это бесконечный упорядоченный набор чисел: X1, X2, X3,... , или обозначается Xn, где n = 1, 2, 3,... .

    **Роль в вычислительных процессах**  
        Последовательности используются для приближённых вычислений: например, для нахождения чисел вроде пи, e, 
        корней уравнений или значений функций — берём всё больше членов последовательности, и результат становится точнее.

    **Предел последовательности**  
        Число a называется пределом последовательности Xn (пишут lim n->беск. (x_n = a)), если члены последовательности при неограниченном росте n становятся сколь угодно близки к a.  
        Формально: для любого e > 0 существует номер N, начиная с которого x_n - a| < e.

    Если предел существует и конечен — последовательность **сходящаяся**.  
    Если предела нет или он +- беск. — последовательность **расходящаяся**.
